AI-Enhanced Coptic Manuscript Analysis (ACMA)

Integrating AI technologies for Coptic manuscript transcription, analysis, and digital editions

Project Overview

The ACMA project aims to develop an integrated framework for computational analysis, transcription, and digital edition of Coptic manuscripts using advanced artificial intelligence. By combining Thoth AI, HTR models, OCR technologies, and linguistic analysis tools, we will significantly reduce manual work while improving accuracy in Coptic studies.
